                                                ***** React Integration *****


1) start the node server.

        > node server


2) open the MySQL Work Bench


3) test the following rest apis by using Postman

        => http://localhost:8080/fetch    (GET)

        => http://localhost:8080/insert   (POST)

        => http://localhost:8080/update   (PUT)

        => http://localhost:8080/delete   (DELETE)


4) install react software

        > npm install -g create-react-app

                - automatically react sotware will install


5) create the react Application

        > npx create-react-app mern-app


6) switch to react Application

        > cd mern-app

7) dowmload axios Module

        - axios module used to connect to server

        > yarn add axios --save

8) create the react application
